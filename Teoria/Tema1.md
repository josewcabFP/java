# JavaDoc

Para generar un java doc se utiliza el siguiente comando:

``` bash
javadoc miarchivo.java
```

Hay que recordar que debemos usar comentarios con la sintanxis:

```java
/**
 * Mis datos 
 * /
 ```
 El archivo generado sera un **html**.
